CONFIGURAÇÃO DO SISTEMA

Para usar o sistema, você precisa:

  1. Criar uma planilha no Google Sheets
  2. Configurar o Google Apps Script
  3. Inserir as URLs abaixo

COMO CONFIGURAR - PASSO A PASSO

1. Criar a Planilha no Google Sheets

Acesse Google Sheets e crie uma nova planilha com duas abas:

  • Aba "Vendas": Colunas: ID, Cliente, DataVenda, TotalLicenca, Entrada, ParcelasQtd, Mensalidade, DataCriacao
  • Aba "Parcelas": Colunas: VendaID, Tipo, Valor, Repasse, Vencimento, Status, NumeroParcela

2. Criar o Google Apps Script

  1. Acesse Google Apps Script
  2. Cole o código do script (veja abaixo)
  3. Salve e implante como aplicativo web
  4. Defina o acesso para "Qualquer um"

3. Configurar URLs

Copie a URL da sua planilha e a URL do script deployado e cole nos campos acima.
